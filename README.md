# VSearch

A simple web browser app for Android, built using Kotlin.

## About

This is a lightweight browser app that lets you browse the internet on your Android device. Built natively using Kotlin for a smooth and simple experience.

## Features

- Simple and easy-to-use browser interface
- Fast browsing experience
- Lightweight app size
- 
## Installation

1. Download the APK from the [Releases](https://github.com/akramsts/vsearch/releases/download/1.0/VSearch.apk) page.
2. Open the downloaded file on your Android device.
3. If prompted, allow installation from unknown sources.
4. Install and open the app.

## Requirements

- Android 7.0 or above
